Working with 3D (c++)

Complex and realistic 3D landscapes created by coding terrain generation algorithms in C++ utilising heightmap meshes and procedural noise.


For this project, I utilized C++ to programmatically generate 3D terrain models. By summing multiple sinusoidal waves and mapping height values to vertices, I was able to create complex and varied terrain meshes. This project demonstrates my proficiency in 3D math, utilising techniques like Perlin noise, height mapping, and vertex manipulation to shape procedural terrains.

I implemented Perlin noise algorithms and sine wave functions in C++ to randomly generate height values. By combining multiple sine waves at different amplitudes and frequencies, I added layers of detail and realism to the terrain. I experimented with different combinations of noise, scales, and sine waves to achieve the visual style I wanted. The wireframe models showcase the terrain topology, while the filled meshes map height to colour for a rendered landscape appearance.

The final landscapes show my ability to create complex 3D terrains procedurally, by coding the mathematical generation of height values and connecting them into dynamic meshes.
