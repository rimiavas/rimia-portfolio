Alien Game

A side scroller game built using p5.js

The Alien Game is a side-scrolling platformer game built with p5.js. The player controls an alien character and must navigate through obstacles to reach the flagpole at the end. This project demonstrates concepts like character animation, physics, collision detection, and winning/losing conditions.

Implementation Section:
The alien character has different animations based on movement direction
Physics concepts like gravity, jumping, and falling are implemented
Obstacles like canyons trigger falling and cause loss of life
Items can be collected for points
Reaching the flagpole triggers a win condition

Key Code Concepts:
Drawing shapes to represent game elements
If/else logic checks for pressed keys to animate character
Character y position changes to create jumping effect
Distance formulas check for collisions with objects
Arrays store data like canyon positions
Boolean variables track game state

This was a fun project that allowed me to explore core coding concepts like animations, physics, and collisions. It provided great practice for game development skills like character control, environmental hazards, and win conditions. In the future, I could expand it by adding more levels, enemies, or powerups.
