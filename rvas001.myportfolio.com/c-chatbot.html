C++ chatbot

An interactive chatbot application built with C++ and openFrameworks.

This project implements a basic chatbot through: Text input field to converse with different bots, Bots with simple logic to respond to user input, Displaying conversation history

Key Features in the chatbot includes: Parrot bot repeats user input, Phrase bot randomly repeats past phrases, Mangle bot generates random words and Markov bot generates responses based on word patterns

Technical Implementation include:Â ofxGui handles text input field and display, Input strings tokenised and passed to bot logic, Bots implemented through simple algorithms: Parrot repeats input, Phrase uses vector to store phrases, Mangle stores words in vector and outputs random ones, Markov uses map to track word relationships, Conversation history shown in scrolling text

This project allowed me to learn: Processing user input in C++, Implementing basic conversational logic, Managing interaction history and display, Working with data structures like vectors and maps. It provides a foundation for more advanced chatbot development in the future.
