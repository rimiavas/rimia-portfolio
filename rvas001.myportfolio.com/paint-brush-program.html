Paint brush program

An interactive paint program allowing users to draw with different brush modes implemented in C++ and openFrameworks.

This project demonstrates my skills in graphics programming using C++ and open Frameworks. Key features include: Multiple brush modes: basic brush, eraser, dynamic brush, rectangle brush. Ability to change brush colour. Drawing is rendered to an FBO for performance. Dynamic brush transforms based on speed and direction. Uses mouse events and interactions to enable painting

I utilised open Frameworks for easy setup of the window and OpenGL context. The main logic is handled in C++ code across ofApp.h/cpp. An FBO is allocated to act as the canvas and is cleared each frame. Different brush modes are implemented as functions that draw primitives. Mouse events toggle painting on/off and pass position to brushes. Key presses cycle through different brush modes. Vector math is used to calculate speed and direction for the dynamic brush.

This project strengthened my C++ abilities along with skills in interactive graphics, UI/UX design, and implementing physics-based effects. The end result is an extensible paint program with a responsive and intuitive feel.
